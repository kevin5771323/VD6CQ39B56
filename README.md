# 校园求职管理系统 java1168

## 项目概述

校园求职管理系统是一款基于SpringBoot和Vue的前后端分离系统。该系统旨在为校园内的求职活动提供便捷服务，主要功能包括个人中心管理、简历管理、职位招聘、通知公告、职位留言、收藏以及面试申请等。

## 技术栈

- 后端：SpringBoot
- 前端：Vue
- 数据库：MySQL
- 开发工具：Idea, vscode
- 插件与框架：mybatis-plus, element

## 功能模块

### 系统角色

- 普通用户
- 企业
- 超级管理员

### 功能列表

- 个人中心
- 简历管理
- 职位招聘管理
- 通知公告
- 职位留言
- 收藏功能
- 面试申请
- 轮播图展示
- 个人后台管理
- 企业管理

## 运行环境

- 后端：JDK 1.8 或更高版本，兼容 SpringBoot 的运行环境
- 前端：Node.js 版本建议 >=10.x
- 数据库：MySQL 5.7 或更高版本

## 部署步骤

1. 搭建开发环境，确保 Java、Node.js 和 MySQL 环境配置正确。
2. 导入 MySQL 数据库文件，构建数据库结构。
3. 使用 Idea 或 vscode 打开后端项目，编译并运行。
4. 使用 Node.js 打开前端项目，编译并运行。
5. 前后端联调测试，确保功能正常。

## 核心亮点

- 前后端分离，支持独立部署，便于维护与扩展。
- 采用 SpringBoot 和 Vue，保证了系统的性能与响应速度。
- 完善的权限管理，满足不同角色的操作需求。
